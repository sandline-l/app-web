# app-web

## 这是实验项目, 
1. 使用了vue, elementui 库
2. 使用axios 来发送请求
3. 使用了router路由, 来跳转
4. 使用了store 来存储公共数据
5. 头部和尾部提取为公共部分
6. 使用了swiper 来写轮播图
7. 静态资源放在public文件夹中
8. 使用 npm run build 来打包为生产文件
